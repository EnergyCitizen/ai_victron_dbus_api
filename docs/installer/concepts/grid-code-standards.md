## What Are Grid Codes?

Grid codes define:
- Acceptable voltage and frequency ranges
- Disconnection/reconnection timing
- Power quality requirements
- Anti-islanding protection

## Why It Matters

Wrong grid code setting causes:
- Frequent disconnections
- Compliance violations
- Grid instability
- Utility complaints
