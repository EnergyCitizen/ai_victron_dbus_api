## Current Limit Types

- **Input Current Limit**: AC input from grid/generator
- **DC Charge Current**: Total charge current to battery
- **DVCC Charge Current**: BMS-controlled limit
- **AC Output Current**: Maximum inverter output

## Why It Matters

Current limits affect:
- Charging speed
- Load balancing in parallel systems
- Battery protection
- Compliance with installation limits
